AWS Student Admin

The aim of this project is to replicate a robust Student Administration System, initially developed for the University of Connecticut's CSE2102 course, Section 3, Group 7. Leveraging AWS cloud-based infrastructure, including DynamoDB, Lambda, and REST API, among others, on the backend, and Vue.js on the frontend, we've created a streamlined and intuitive user interface. This project represents a significant milestone in my journey, showcasing not only my technical skills but also my ability to effectively tackle complex challenges and deliver results.
